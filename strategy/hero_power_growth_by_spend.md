# 英雄戦力育成方針: 課金帯別

作成日: 2026-05-05

ラストウォーの英雄戦力を、無課金・微課金・廃課金でどう伸ばすかの整理。主な根拠は、公式Discord `攻略tips > 育成相談コーナー` の相談ログ走査と、公開Webガイドで確認できる専用武装・装備仕様。

この文書は、個人ごとの育成相談で「どこに資源を入れるべきか」を判断するための基準として使う。生ログの転載ではなく、相談傾向を実務方針に要約している。

## 結論

英雄戦力は、英雄の星・スキル・専用武装だけでは伸び切らない。実戦では、装備、研究、ドローン、兵種専門、配置、相手兵種まで揃って初めて勝率に出る。

課金が少ないほど、方針は単純にする。

- 無課金: 一軍一点集中。染め完成を急がず、単体でも混成でも使える英雄を優先。
- 微課金: 一軍完成を最優先。兵種乗り換えは、既存投資を失っても回収できるかで判断。
- 廃課金: 複数編成、武装30、MR、装飾Lv4+、相手別入替まで投資対象。ただし順番を誤ると費用対効果は落ちる。

## 共通原則

### 一軍特化が基本

このゲームは、一軍の完成度が最も効く。二軍・三軍に手を出すのは、一軍の星5、主要スキル、主要装備、主要武装20が見えてからでよい。

特に無課金・微課金は、二軍の見栄えより一軍の勝率を優先する。上級アリーナの英雄総戦力を上げたい場合でも、まず一軍を崩さない。

### 専用武装は20が基準

専用武装の節目は `1 / 10 / 20 / 30`。相談ログと公開ガイドの両方で、初期目標はLv20が妥当という扱い。

- Lv1: 解放だけでも効果がある
- Lv10: 中間目標
- Lv20: 実戦上の主要目標
- Lv30: コストが重く、廃課金・上位帯向け

Lv20からLv30は、0から20にするより重い。微課金以下は、1体を30にするより主力複数を20に広げる方が基本的に強い。

### 装備は中途半端に止めない

装備の目安は `0 / 20 / 40 / 星2 / 星4 / MR`。10、30、星1、星3で止めるより、次の節目まで届く対象を選ぶ。

初期の赤装備は火力役のレールガンが優先されやすい。ただし、DVA前衛やロケランの盾役など、前衛が落ちることで負ける場合はレーダー・装甲の価値が上がる。

### 研究は「科学が常に上」ではない

研究、装備、武装は固定順位ではなく、同じコストで勝率がどれだけ上がるかで見る。

ただし、英雄研究の全英雄ステータス系は費用対効果が高い。攻撃、防御、HPの優先では、PvP目線なら攻撃が最も評価されやすい。

## 無課金

### 目標

現実的な目標は、サーバー環境によって中級上位から上級アリーナ下限。上位サーバーでは、上級入りだけでもかなり厳しい。

### 方針

一軍だけを見る。二軍・三軍、武装30、広い混成、途中からの兵種乗り換えは原則避ける。

ロケランは無課金でも可能。ただし、英雄研究、日々の資源回収、スキルメダル、武装かけら、装備が遅れると完成がかなり遅い。いきなりロケラン染めを目指すより、単体でも混成でも使える英雄を先に育てる。

優先候補:

- アダム: 戦車・航空・ロケラン移行の選択肢を広げる
- スウィフト: 単体性能が高く、航空相手にも強い
- ルシウス: 戦車/航空/ロケランの橋渡しで使いやすい
- キンバリー: 戦車継続なら火力軸
- マーシャル/ウィリアムズ: 戦車継続時の土台

### やらないこと

- 一軍の主要武装20前に武装30へ行く
- 二軍武装20を先に作る
- 専門研究に入った後で兵種を大きく変える
- 流行だけで航空からロケランへ乗り換える

## 微課金

### 目標

パス購入、月数千円から1.5万円程度なら、一軍完成度で中級上位から上級下限を狙う。上級安定やTop50はサーバー強度に強く左右される。

### 方針

一軍の完成を優先しつつ、将来の兵種移行に備えて汎用性の高い英雄を育てる。

S2後半からS3で航空からロケランへ移行する場合、数か月は自分より課金額の少ないロケラン勢にも劣る可能性がある。移行中はアリーナ報酬も落ちやすい。すでに航空の武装20、チップ育成、専門研究が進んでいるなら、そのまま伸ばす方が無難な場面が多い。

### 投資優先

1. 一軍英雄を星5へ
2. 一軍主要スキルを上げる
3. 主要専用武装をLv20へ
4. 火力役の赤装備、前衛の防御装備を節目まで上げる
5. 英雄研究、兵種研究を偏りすぎない範囲で進める
6. 余力で二軍の基礎を作る

専用武装かけらは、次の新英雄/武装に備えてある程度残す。相談ログでは、微課金が全て使い切らず、次の主要英雄に備える判断が繰り返し推奨されていた。

### 課金先の考え方

高効率パック、パス、イベント小額課金、赤設計図、スキルチップ素材、ドローン部品が候補。ドミネーター実装初期のように、短期で一軍戦力が大きく伸びる要素は、基地スキンより優先されることがある。

## 廃課金

### 目標

Top10、Top20、越境上位戦での勝率。単純な戦力値より、相手別の編成変更、装備付け替え、配置差が重要になる。

### 方針

一軍の主要英雄を武装20で揃えた後、武装30、複数兵種、MR、装飾Lv4+、ドローン、戦術カード、ドミネーターまで広げる。

ただし、廃課金でも順番はある。主要英雄の20が揃う前に30へ突っ込むと、選択肢が狭くなる。

### 投資対象

- 専用武装30: 上位帯では必要になるが、最初から全員30を前提にしない
- 複数編成: ロケラン純正、ロケラン+ルシウス、ロケラン+マーフィ、戦車4+1など
- MR装備: 火力軸のレールガン、防御軸のレーダー/装甲
- 装飾Lv4+: スキルダメージ増加、被ダメージ軽減、会心ダメージ増加の順で評価されやすい
- ドローン/ドミネーター: 基礎ステータスがあるほど効果が出る

## 兵種別の判断

### 戦車

無課金・微課金が継続しやすい。キンバリー軸で土台を作りやすく、S3以降も需要は残る。

S4環境では、ウィリアムズを抜いてアダム/スカーレットを入れる4+1型も見られる。対戦車ではルシウスを入れる判断もある。

### 航空

S3以降はやや苦しい時期があるが、S4以降にマーフィ混成などで再評価される場面がある。DVA武装20や航空チップ育成が進んでいる場合、安易なロケラン移行は損失が大きい。

DVA前衛は成立するが、装備と相手次第。ルシウスは生存させたい支援役なので、前衛固定ではなく後衛配置も検討する。

### ロケラン

航空に強く、格上航空にも勝負できる。ただし戦車相手は不利が残る。ロケラン移行中は戦力の伸びが一時的に落ちるため、無課金・微課金は覚悟が必要。

純正ロケランだけでなく、ルシウスやマーフィを入れる混成も使われる。アダム、スウィフト、テスラ、マクレガー、フィオナの育成順は、手持ち武装と相手環境で変える。

## 英雄メモ

| 英雄 | 見方 |
|---|---|
| ルシウス | 戦車/航空/ロケランの橋渡し。S4で武装20を急ぐ価値が高い場面あり |
| スウィフト | 単体性能が高く、混成でも使いやすい。航空相手の圧力が強い |
| アダム | 兵種移行を迷う人の選択肢を広げる。武装実装後は主軸化する例もある |
| テスラ | ロケラン火力軸。余裕があるなら20優先の場面が多い |
| マクレガー | ロケラン前衛/反撃役。武装20は有用、30は上位帯向け |
| フィオナ | 採用するなら火力装備とスキルを優先。レールガン/チップを持たせる判断が多い |
| マーフィ | 航空+マーフィや戦車/防衛で有効。ただし武装だけでなく防御装備も必要 |
| キンバリー | 戦車火力軸。戦車継続なら20以降も候補 |
| DVA | 航空火力軸。前衛運用は装備と耐久が前提 |

## 判断フロー

1. いま一番育っている兵種は何か
2. その兵種の主要英雄は星5か
3. 主要武装はLv20まで届いているか
4. 主要スキルは実戦で足を引っ張っていないか
5. 火力役の赤装備、防御役のレーダー/装甲は節目まで届いているか
6. 兵種専門研究に入っているか
7. 乗り換えた場合、何か月弱くなるか
8. その間のアリーナ報酬低下を許容できるか

微課金以下で、3から6がかなり進んでいるなら、基本は継続。まだ進んでいないなら、アダム/スウィフト/ルシウスのような混成でも使える英雄を経由して選択肢を広げる。
