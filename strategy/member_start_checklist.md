# S6 Start Checklist for Members

JDX連盟員向け。S6開始前から開始直後にやることだけを抜き出した短縮版。

## 共有方針

開幕直後は、敵を攻めるより先に#534内の内政基盤を作る。

最初に差が出るのは、戦闘よりも以下の取り漏らし。

- 職業スキルの振り直し
- 胞子工場と真菌研究所
- 初回撃破報酬
- 漁場確保と釣り
- 魚寄付と陣営科学
- 軍功とデイリー報酬
- 戦術カードと耐性バフの確認
- ブラッツUR化など、実行タイミングを間違えやすい育成

## Discord貼り付け用

```text
【S6開始前・開始直後チェック / JDX】

今回のS6は、開幕直後の取り漏らしで差が出ます。
最初は敵を攻めるより、#534内の都市・拠点・漁場・施設育成を優先します。

■ 開始前にやること
1. 職業スキルを確認
　必要ならリセットし、S6スキル用ポイントを残してください。
　作戦経験、建造、軍功還元、爆釣タイム、特級釣り餌、無料高級移設などが候補です。

2. ブラッツUR化は一旦待機
　シーズンイベント英雄交換でVSポイントを稼げる可能性があります。
　開幕前に上げ切らず、指示を待ってください。

3. 重要資源は勝手に使い切らない
　レア餌、高級移設、伝説再臨チケット、シーズン資源、加速は、開幕後の画面とイベント対象を確認してから使ってください。

4. Discordと連盟チャットを見られる状態にする
　漁場、都市、拠点、集結の指示はチャットで流します。

■ 開始直後にやること
1. 胞子工場を開放・強化
　ジャングルきのこを使い、胞子生産を早めに伸ばしてください。

2. 真菌研究所でウイルス耐性を上げる
　耐性が低いと、終末精鋭、森の見張りゾンビ、漁場攻略に参加しにくくなります。

3. 倒せるS6モンスターを消化
　終末精鋭、森の見張りゾンビ、Shadow Breakerを自分の耐性で倒せる範囲から進めてください。
　初回撃破報酬を取り漏らさないことが大事です。

4. 軍功を確認
　軍功は累計値です。軍功アイテムを使っても総軍功は減りません。
　階級、デイリー報酬、交換所を毎日確認してください。
　初回撃破、都市/釣り場初占領、魚寄付、陣営科学寄付は取り漏らさないでください。

5. 漁場集合に参加
　漁場が開いたらR4/R5の指示に従って参加してください。
　勝手に別の漁場や拠点へ進まないでください。

6. 釣り開始後は魚を寄付
　魚は個人で貯め込まず、陣営科学や連盟スキル用の寄付を優先します。

7. 釣り餌を毎日確認
　通常餌は訪問者から受け取り。
　金/特級釣り餌は職業スキルのCTを忘れず回してください。画像上は23時間、既存メモは約23.5時間です。

8. 巨大ワニが出たら報告
　巨大ワニは1時間で消えます。
　出たら連盟チャットで共有し、集結で倒してください。
　魚籠報酬は1日10個回収が目安です。

■ やらないこと
・R4/R5指示なしに都市、拠点、漁場へ単独進軍しない
・#476方面へ深追いしない
・中央漁場をWeek 1から主目標にしない
・魚を全部個人用に貯め込まない
・ブラッツUR化を開幕前に上げ切らない
・レア餌、高級移設、重要資源を確認なしに使い切らない

Week 1の目的は、#534内の基盤を固め、Week 2以降の中央・盟約・祭壇に備えることです。
```

## 開始前チェック

| 優先 | やること | 対象 | メモ |
|---:|---|---|---|
| 高 | 職業スキルを確認する | 全員 | S6スキル用ポイントを残す。軍功還元、特級釣り餌、黄金魚宝箱、領土内攻防、爆釣タイムを役割別に見る |
| 高 | ブラッツUR化を待機する | 対象者 | シーズンイベント英雄交換とVSポイント確認後に実行 |
| 高 | Discord/連盟チャットを確認できる状態にする | 全員 | 開幕指示を見逃さない |
| 中 | 伝説再臨チケットを確認する | 保有者 | 開幕後のシーズン資源獲得に使えるか実画面で確認 |
| 中 | レア餌・高級移設・重要資源を温存する | 全員 | 使う前にイベント対象と連盟方針を確認 |
| 中 | シーズンパス/週間配達を確認する | 購入者 | 買う人は序盤の耐性上げに効くか確認 |

## 開始直後チェック

| 優先 | やること | 理由 | 注意 |
|---:|---|---|---|
| 高 | 胞子工場を開放・強化 | 胞子生産量が以後の成長を決める | ジャングルきのこを使う |
| 高 | 真菌研究所を強化 | ウイルス耐性を上げる | 胞子を貯め込まず耐性へ回す |
| 高 | S6モンスター初回撃破を進める | 軍功、素材、シーズン資源 | 無理な高Lvより倒せる範囲を確実に |
| 高 | 漁場集合に参加 | 初日から魚寄付を回す | 指示なしの単独行動は禁止 |
| 高 | 釣りを始める | 陣営科学と連盟スキルの燃料 | 魚を個人倉庫に貯め込まない |
| 高 | 軍功を確認 | 階級、デイリー、交換所に関係 | 軍功は累計。初回撃破/占領と寄付を取り漏らさない |
| 中 | 通常餌を受け取る | 毎日の積み上げ | 訪問者を確認 |
| 中 | 金/特級釣り餌を生成/使用 | 画像上はCD23時間、既存メモは約23.5時間 | 職業スキルの押し忘れ注意 |
| 中 | 巨大ワニを報告 | 1時間で消える | 集結、魚籠報酬回収 |
| 中 | 戦術カードと耐性バフを確認 | モンスター/戦闘効率に関係 | カードパック、不要カード分解、キノコ/週間配達などの耐性表示を確認。キノキノ守れは半径3マス、耐性+250、CD24h、同種非重複 |

## 役割別チェック

| 対象 | 最初にやること |
|---|---|
| 高戦力 | 漁場/都市/拠点集合、S6モンスター高Lv処理、巨大ワニ集結 |
| 中堅 | 耐性上げ、釣り、魚寄付、軍功回収、集合参加 |
| 低戦力/サブ | 釣り、魚寄付、連盟ヘルプ、倒せる範囲の初回撃破 |
| 課金者 | シーズンパス/週間配達/耐性補助を開幕後に確認し、イベント対象を見て使う |
| 無課金/微課金 | 毎日ログイン、餌回収、魚寄付、軍功、耐性上げを取り漏らさない |

## R4/R5が同時に出す指示

- 最初に集合する漁場
- 都市・拠点・漁場で単独行動しないこと
- 魚寄付先の優先順位
- 巨大ワニ発生時の報告先
- キノコ小屋耐性の発動担当
- 祝福された魚を使うタイミング
- 役職連盟スキルの発動者と代理
- ブラッツUR化や伝説再臨チケットの実行タイミング
