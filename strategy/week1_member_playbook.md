# Week 1 Member Playbook

S6開始後1週目に、JDX連盟メンバーへ共有する実行メモ。

## 基本方針

Week 1は、敵サーバーへ大きく攻める週ではなく、#534内の内政基盤を作る週と考える。

優先するもの:

1. #534内の都市・拠点・漁場を確保する
2. 胞子工場と真菌研究所を育て、ウイルス耐性を上げる
3. 終末精鋭、森の見張りゾンビ、Shadow Breakerで軍功と素材を取る
4. 釣りを開始し、魚寄付で陣営科学と連盟スキルを進める
5. 軍功階級とデイリー報酬を取り漏らさない
6. #440の全体方針と今後の盟約判断に備えて、陣営科学寄付を進める
7. 開幕前後の職業スキル、釣り餌、伝説再臨チケット、ブラッツUR化のタイミングを間違えない

Week 1は自戦域内移設が中心。中央漁場や他戦域展開はWeek 2以降の準備として扱う。

## Discord貼り付け用

```text
【S6 Week 1 / メンバー向け行動方針】

Week 1は、敵を攻めるより先に#534内の基盤作りを優先します。
各自、以下を毎日確認してください。

1. 開幕前に職業スキルを確認
　必要ならリセットし、シーズンスキル用ポイントを残して振り直してください。作戦経験、建造、軍功還元、爆釣タイム、特級釣り餌、無料高級移設などが候補です。

2. 胞子工場を上げる
　ジャングルきのこを使い、胞子生産を早めに伸ばしてください。

3. 真菌研究所でウイルス耐性を上げる
　耐性が低いと終末精鋭、森の見張りゾンビ、漁場攻略に参加しにくくなります。

4. 終末精鋭・森の見張りゾンビ・Shadow Breakerを消化
　軍功、素材、耐性上げに関係します。初回撃破報酬を取り漏らさないでください。

5. 漁場確保と釣りに参加
　漁場が開いたら連盟指示に従って参加してください。釣った魚は個人で貯め込まず、陣営科学や連盟スキル用の寄付を優先します。

6. 軍功を毎日確認
　軍功は累計値です。軍功アイテムを使っても総軍功は減りません。階級、デイリー報酬、交換所を毎日確認してください。

7. 都市・拠点・漁場の行動は指示に従う
　勝手な単独行動で接続や防衛線が崩れると、後半の祭壇・前哨基地・都市破壊に影響します。

8. レア餌・高級移設・重要資源は使う前に確認
　Week 2以降に必要になる可能性があります。迷ったらR4/R5に確認してください。

9. ブラッツUR化は開幕前に上げ切らない
　シーズンイベント英雄交換でVSポイントを稼げる可能性があるため、実行タイミングは指示を待ってください。

Week 1の目的は、#534内の都市・漁場・軍功・魚寄付を取り切り、#440の全体方針が出た後にWeek 2以降の中央展開へ合わせられる状態を作ることです。
```

## Day 1チェック

| 優先 | やること | 理由 | 注意 |
|---:|---|---|---|
| 高 | 胞子工場を開放・強化する | 胞子生産量が以後の成長速度を決める | 軍功交換所の表示条件にも関係する可能性 |
| 高 | 真菌研究所を強化する | ウイルス耐性を上げ、S6モンスターや漁場攻略に参加しやすくする | 胞子を貯めすぎず、耐性上げに回す |
| 高 | 職業スキルを振り直す | S6スキル用ポイントと初動効率に影響する | 開幕前にリセットし、必要ポイントを残す |
| 高 | 終末精鋭を倒す | 初回撃破、軍功、素材の基礎 | 自分の耐性で倒せる範囲から進める |
| 高 | 森の見張りゾンビを倒す | 初回撃破と軍功に関係 | 無理な高Lv狙いより確実に消化 |
| 高 | Shadow Breakerを消化する | ジャングルきのこ、軍功、耐性素材に関係 | 回数や報酬条件を確認 |
| 高 | 漁場開放時間を確認する | 初日から漁場2つ確保が重要 | 集合指示があれば優先して参加 |
| 高 | 釣りを始める | 魚寄付と陣営科学の基盤 | 魚を個人倉庫に貯め込まない |

| 中 | 伝説再臨チケットを確認する | シーズン資源の獲得導線になる | 実画面の交換先を確認してから使用 |
| 中 | 連盟チャット/Discordを確認する | 漁場・都市・拠点の集合指示を見逃さない | 単独で先走らない |

## Week 1 毎日やること

| 項目 | やること |
|---|---|
| 施設 | 胞子工場、真菌研究所を優先して上げる |
| 耐性 | 自分が倒せるS6モンスターLvを毎日更新する |
| 軍功 | 軍功獲得、階級、デイリー報酬、交換所を確認する |
| 釣り | 餌を消化し、魚寄付または連盟方針に従って使う |
| 金の釣り餌 | 職業スキルのCT約23.5hを見て、毎日生成/使用を忘れない |
| 漁場 | 連盟が取った漁場で釣る。他連盟漁場は徴収がある可能性を理解する |
| 巨大ワニ | 発生したら1時間以内に集結で倒す。魚籠報酬は1日10個回収を目安にする |
| 都市/拠点 | R4/R5の指示に従って参加。勝手に進軍先を変えない |
| 資源 | レア餌、高級移設、重要な交換アイテムは使う前に確認 |
| 情報 | Discord、連盟チャット、メールを確認する |

## やらない方がよいこと

- R4/R5指示なしに、都市・拠点・漁場へ単独で突っ込む
- 魚を全部個人用に貯め込み、陣営科学や連盟スキルを止める
- レア餌や高級移設を理由なく消費する
- ブラッツUR化を開幕前に上げ切る
- 職業スキルの振り直しを忘れてS6ポイントを足りなくする
- 中央漁場をWeek 1から主目標にする
- #476方面へ深追いして兵を消耗する
- 盟約が解放される前に、外交だけで動けると誤解する

## メンバー別の動き

| 対象 | Week 1の役割 |
|---|---|
| 高戦力メンバー | 漁場・都市・拠点の集合参加、S6モンスター高Lv処理、連盟指示時の主力 |
| 中堅メンバー | 耐性上げ、釣り、魚寄付、軍功回収、集合時の人数確保 |
| 低戦力/サブ | 釣り、魚寄付、軍功デイリー、連盟ヘルプ、無理のないモンスター消化 |
| 課金する人 | 覚醒、キノコ耐性、餌、施設加速は連盟方針とイベント対象を確認してから使う |
| シーズンパス/週間配達を買う人 | 序盤の耐性上げに影響するため、買うなら開幕直後に効果を確認して使う |
| 無課金/微課金 | 毎日ログイン、釣り、魚寄付、軍功、耐性上げを取り漏らさない |

## Week 1の成功条件

- #534内の主要都市・漁場・拠点を押さえている
- メンバーが胞子工場と真菌研究所を理解している
- 魚寄付と陣営科学寄付が毎日回っている
- 軍功階級と報酬の取り漏らしが少ない
- #440方針を確認する体制があり、#509方面の候補情報が整理されている
- #476方面で無駄な消耗をしていない
- Week 2以降の中央漁場、盟約、聖所、祭壇準備へ移れる

## 要確認

- 初日漁場開放時間と実際に取れる漁場数
- 中央エリア漁場が本当にWeek 1占領不可か
- 軍功交換所の正確な表示条件
- 職業スキル、ゴールド餌、レア餌の最適な使いどころ
- キノコ耐性の重複/加算仕様
