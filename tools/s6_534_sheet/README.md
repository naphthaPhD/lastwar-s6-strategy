# S6#534 Google Sheet 管理スクリプト

`S6#534` スプレッドシート本体に貼る軽量 Apps Script。

## 目的

- `管理表たたき` を正本にする。
- `マップ表示テンプレ` の配置を使って `全体マップ` を再描画する。
- 所有連盟・種別・破壊状態を `全体マップ` の文字色へ反映する。

## メニュー

スプレッドシートを開くと `S6#534管理` メニューが出る。

| メニュー | 内容 |
|---|---|
| `全体マップ更新` | `管理表たたき` から `全体マップ` を更新する |

## 色ルール

| 対象 | 文字色 |
|---|---|
| #534 所属 | 青 |
| #509 / #440 / #511 所属 | 緑 |
| #503 / #480 / #523 / #476 所属 | 赤 |
| 交易地 | 黒 |
| 破壊 | 灰 |

## 導入

1. `S6#534` を開く。
2. `拡張機能` -> `Apps Script` を開く。
3. `Code.gs` をこのフォルダの `Code.gs` で置き換える。
4. 保存する。
5. スプレッドシートを再読み込みする。
6. `S6#534管理` -> `全体マップ更新` を実行する。

## タイムアウト対策

`全体マップ更新` は、セルごとの書き込みではなく配列を作って一括反映する。もし Apps Script で「起動時間の最大値を超えた」またはタイムアウト系のエラーが出る場合は、シート側の Apps Script に旧版が残っている可能性が高い。最新の `Code.gs` に貼り替えてから再読み込みする。

## 前提タブ

- `管理表たたき`
- `マップ表示テンプレ`
- `全体マップ`
- `連盟盟約状況`

`連盟盟約状況` の連盟A/B所属サーバ列から、連盟タグの所属サーバを補助判定する。未登録タグは、所有しているエリアのサーバを補完値として使う。
