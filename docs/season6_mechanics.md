## 前哨基地争奪戦 画像確認

2026-05-03にユーザー添付のゲーム内画像 `IMG_6876.png`、`IMG_6877.png`、`IMG_6879.png`、`IMG_6880.png`、`IMG_6881.png` を確認。以下は画像上で読めた内容。

### 配置・建設

- 大統領のみ、各配置段階で自陣営に所属する戦域の指定エリアへ前哨基地を1カ所配置できる。
- 前哨基地配置後、該当戦域に所属する連盟が建設する必要がある。
- 前哨基地の位置は変更できない。
- 建設完了後の前哨基地は、その戦域に所属する全ての連盟の領土として見なされる。
- 建設に最も貢献した連盟が前哨基地の領主となる。
- 配置画面では `前哨基地を配置(1/4)` と表示され、全4段階の配置がある。

| 段階 | 画像上の解放時間 |
|---|---:|
| 第1前哨基地 | 15:52:56後 |
| 第2前哨基地 | 7d 15:52:56後 |
| 第3前哨基地 | 14d 15:52:56後 |
| 第4前哨基地 | 21d 15:52:56後 |

表示はスクリーンショット時点の相対時間。実運用では、初回解放後に7日刻みで第2-第4前哨基地の配置日が来る前提でカレンダー化する。2026-05-04の公式Discord補足では、前哨基地は4週目に配置・建設開始で、2週目時点には存在しないと説明されていた。

### S6 City Clash 公式ルール画像

2026-05-13から2026-05-14のユーザー添付ゲーム内画像で、`S6 City Clash` と Deepwood系都市/拠点のルール説明を確認した。都市譲渡、放棄、E列/C列の取得タイミング判断に直接関係するため、読めた範囲をここに固定する。

## 公式Wiki貼り付けで確定した詳細

### 移設・戦域移動

- Week 1 は自戦域内のみ移設可能。
- Week 2 以降は `高級移設` で他戦域へ移動可能。
- `連盟遠征集結地点` を設置すると、`連盟移設` で他戦域へ移動可能。
- 元の戦域へ戻る場合も移設アイテムが必要。
- 土曜の連盟対決/戦争日には特殊移設ルールがある。

### 盟約

- 条件: 同一シーズン、同一陣営、領地が1つ以上隣接、双方クールダウンなし、水曜/土曜ではない、盟主または理事が申請、双方確認。
- 同時に保持できる友好連盟は1つのみ。
- 盟約後は友好連盟の領地を隣接元として領土拡大できる。
- 自鯖/自陣営の都市・漁場も盟約締結の隣接元になる見込み。敵陣営都市は破壊扱いのため隣接元にならない。
- 領地が分断されると盟約は即時解除。第三者攻撃による分断ではクールダウンなし。
- 盟約成立後は3日間クールダウン。自主解除ではクールダウンなし。
- 盟約解除時は援軍が自動帰還し、友好連盟グループチャットが閉鎖される。

### 釣り

- 所属陣営の連盟が占領した漁場のみ釣り可能。
- 魚種は漁場の初期所属陣営とレベルで決まり、現在の占領陣営では変わらない。
- タイミングバーは魚種確率と寄付エネルギーに影響せず、魚の重さに影響する。
- 同じ魚種は重さに関係なく同じエネルギーを提供する。
- 同陣営の他連盟漁場で釣ると徴収が発生する場合がある。徴収魚は個人倉庫に入らないが、図鑑と魚種順位にはカウントされる。

### 祭壇

- 毎週火曜サーバー時間12:00から1時間。
- 連盟領土との隣接不要。祭壇所在戦区に入り、保護状態でなければ派兵可能。
- 1連盟の保有上限は3つ。
- 2026-05-17日本語HTML更新では、3週目火曜にLv1とLv2を同日2カ所取れるか、1日1カ所だけかという質問あり。2026-05-20日本語Discord書き出し画像では、同じレベルの祭壇を複数占領できた実地共有あり。少なくとも1日1祭壇限定ではなさそうだが、Lv違い同日占領は実画面で確認する。
- 2026-05-20日本語Discord書き出し画像では、祭壇は汚染地域外からも占領可能、占領時の滞留部隊は最大20名、占領完了まで20分という実地共有あり。
- 参加報酬や直接軍功は特になさそう。占領後、R4以上が `大河氏族の贈り物` を1回召喚し、祭壇周辺の魚群を連盟員がクリックして漁獲を得る。巨大ワニ魚箱とは別上限で、取得は10個制限らしき表示あり。
- `とぐろの祭壇` と `裂風の祭壇` は同時保有不可。
- `こだまの祭壇` と `彩羽の祭壇` も同時保有不可。
- 同一スキル祭壇を複数保有してもスキル効果は1回のみ。
- 祭壇廃棄には60分かかり、争奪開始1時間前以内は自主放棄不可。

### 英雄覚醒

- S6対象例: 第1期キンバリー、第2期DVA、第3期テスラ。
- 解放条件: 英雄星5、専用武装Lv20、戦域がS6に入っていること。
- 解放には専用覚醒かけらを消費。強化には専用覚醒かけらと汎用覚醒かけらを使用可能。
- 覚醒後、第4スキルは覚醒スキルに置換され、元の第4スキル効果は覚醒スキルに統合される。

### 前哨基地

- 大統領のみ、各配置段階で指定未配置エリアに1つ配置可能。
- 聖所/議事堂周辺の4カ所は初期未建設。初回は1カ所のみ建設可能で、以後1週間ごとに1カ所ずつ追加される。
- 自戦域への配置は必須ではないという公式Discord会話あり。さらに、同陣営であれば他戦域の前哨基地を建設できるという補足もある。自陣営内の他戦域への足掛かり、中心防衛、議事堂応援ルートも配置目的に含める。
- 配置後の位置変更不可。破壊済みエリアや既存前哨基地エリアには再配置不可。
- 建設は本戦域の連盟が行う。建設貢献1位の連盟が所有連盟。
- 建設完了後、本戦域すべての連盟に領土リンク効果を提供。
- 敵前哨基地攻撃には、目標前哨基地に隣接する領土、または周辺都市/拠点を占拠した接続ルートを確立する必要がある。
- 本戦域のいずれかの連盟が目標前哨基地と接続できていれば、本戦域の全連盟が攻撃に参加できる見込み。
- 接続がない場合は、周辺拠点を占領している味方陣営連盟と盟約を結び、連携攻撃する必要がある見込み。
- 前哨基地争奪戦は決められた時間帯のみ発生する模様。配置と争奪の順番は見えたが、水曜23時/木曜11時表示の関係は要確認。
- 自戦域前哨基地は自戦域中心を全連盟で防衛するためのもの、敵前哨基地破壊は敵陣営中心の防衛を崩すためのものというモデレーター整理あり。
- 攻撃側が占領進行度100%に到達すると永久破壊。100%未達なら防衛成功で所有者変更なし。
