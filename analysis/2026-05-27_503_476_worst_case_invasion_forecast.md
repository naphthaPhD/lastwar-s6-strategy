# 503 Pact Rebuild Worst-Case Invasion Forecast

## 1. Executive summary

盟約が結び直され、#503側連盟も#534を攻撃できる前提では、#534の最悪パターンは「#476B/#476Cが既存足場から都市を脅し、同時に#503側へつながる右方向の漁場ラインを開く」形である。

この場合、#534の拠点変化は次のように進む。

1. #476Cが `A-7 / B-7 / C-7 / C-9` から上・内側の都市を脅す。
2. #476Bが `C-11 / D-7` から `D-11 / D-9 / E-7` を狙う。
3. `D-11` が落ちると `c-12(JDX)` と `d-12(JDX)` が同時に危険化する。
4. `D-9` が落ちると `d-10(SHA)` が開き、右方向への曲がり角が成立する。
5. `E-7` まで落ちると、#503側合流ルートの準備が現実化する。

結論として、都市を全部守るのではなく、`D-11`、`D-9`、`E-7` の3点拒否を最優先にする。

## 4. Worst-case timeline

### Phase 1: 既存足場から都市を脅す

追加漁場を取らなくても危険な都市候補。

| 都市 | 所有 | 主な圧力元 | 評価 |
|---|---|---|---|
| c-12 | JDX | 476B C-11 | 最重要。JDX接続の中核 |
| c-10 | SHA | 476B C-11 / 476C C-9 | 重複圧力が強い |
| c-8 | SHA | 476C C-7/C-9 / 476B D-7 | 重複圧力が強い |
| b-10 | SHA | 476C C-9 / 476B C-11 | 釣り対象になりやすい |
| b-8 | MOE | 476C B-7/C-7 | 上側撹乱 |
| b-6 | MOE | 476C B-7/C-7 | 上側撹乱 |

### Phase 2: 右方向の曲がり角を作る

ここで#534の拠点変化が決定的になる。

| 取られる漁場 | 起きる変化 | 最悪度 |
|---|---|---|
| D-11 | `d-12(JDX)` と `d-10(SHA)` が開く。`c-12(JDX)`と合わせてJDX二面圧力になる | 最高 |
| D-9 | `d-10(SHA)`が開き、右方向への曲がり角が成立する | 高 |
| E-7 | `e-6(4tH)`、`e-8(SHA)`が開き、Dラインを越えて深部へ入られる | 高 |
| C-13 / D-13 | JDX右側へ広げられ、#503側合流準備になる | 中-高 |
| E-9 / E-11 / E-13 / F-13 | 土曜以降の右側・#503側合流帯になる | 中-高 |

### Phase 3: #503側合流

最悪形では、#476Bが右方向、#476Cが上・内側、#503側が右上から圧力を重ねる。これにより#534は、都市防衛、漁場即応、右側監視を同時に求められる。

この段階になると、#534の前線はCラインからD/Eラインへ押し込まれ、SHA/JDX/MOEの都市群が「守る都市」ではなく「敵が防衛力を吸う餌」になる。
